### MVC
**Model-View-Controller**   
**Model = data** - 데이터베이스   
**View** - 화면 디자인   
**Controller** - 화면 디자인과 데이터베이스를 합쳐줌   

웹 서버에서 실행되는 프로그램인 서블릿이 응답하게 해줌. _서블릿 = 객체_

**프론트** - 백엔드에게 요청해서 받은 것을 완성시키는 것    
**백엔드** - 사용자의 요청을 받아 서버를 처리하는 것    
이렇게 완성된 것이 html이고 그것을 사용자 브라우저에 띄워준다.
프론트의 서버 주소만 아라면 되고 프론트는 백앤드, 백앤드는 데이터베이스의 서버만 알면 된다.
